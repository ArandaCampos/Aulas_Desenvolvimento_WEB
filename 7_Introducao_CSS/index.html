Introdução ao CSS

Cascading Style Sheets - Folhas de estilização em cascata

 Para que server?

Para alterar todos os aspectos visuais (cor, tamanho, formato, posição, animações, etc...)

 Qual arquivo é css?

Todo arquivo css termina com a extensão .css

 Como lincar o css?

No arquivo com extensão .html, dentro da tag <head> com a tag <link>

 Seletores

Os seletores não muito parecidos, então fique atento.

.nome ≠ #nome ≠ nome

 Identificadores

Seletor por identificaçao (ID) deve iniciar com jogo da velha

Ex: #nome-do-id

 Ponto de atenção IV

Seletor por tag afeta todos os elementos que tenham aquela tag

Como inserir minha estilização na página?



<link href="style.css" rel="stylesheet" type="text/css">
