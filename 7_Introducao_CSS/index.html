Introdução ao CSS

Cascading Style Sheets - Folhas de estilização em cascata

 Para que server?

Para alterar todos os aspectos visuais (cor, tamanho, formato, posição, animações, etc...)

 Ponto de atenção I

A sintaxe (modo que escrevemos) é diferente do HTML.

 Ponto de atenção II

Seletor de classe deve iniciar com ponto final

Ex: .nome-da-classe

 Ponto de atenção III

Seletor por identificaçao (ID) deve iniciar com jogo da velha

Ex: #nome-do-id

 Ponto de atenção IV

Seletor por tag afeta todos os elementos que tenham aquela tag

 Ponto de atenção V

Os seletores não muito parecidos, então fique atento.

.nome ≠ #nome ≠ nome

Sintaxe do CSS

Seletores

Define quais elementos serão modificados.

Como selecionar um elemento HTML


<h1 class="vermelho" id="red">Texto vermelho</h1>
  • h1 {         
       color: red;
     }           

    Por tag

  • .vermelho {  
      color: red; 
     }           

    Por classe

  • #red {       
      color: red; 
     }           

    Por Id
