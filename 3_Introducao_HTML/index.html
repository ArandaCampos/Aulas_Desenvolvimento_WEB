 Semântica

 Semântica é o estudo do significado!

 O que significa tag?

 Tag significa marcação em português. Ela é representada pelos sinais de menor e maior.
< >

 Tim Berners-Lee - de novo!

 Criado em 1990, o HTML é uma linguagem de marcação, que deve ser lido e interpretado pelos navegadores. Hoje, o HTML está na sua versão 5, também conhecida como HTML Semântico.

 O que faz uma linguagem de marcação?

 Armazena conteúdo de mídia (texto, imagem, vídeos, links) e seu sentido (semântica).

 Como escrevo no .html?

 Tag significa marcação em português e, é com ela que escrevemos o HTML.

 Como meu .html é lido e aparece na tela?

Tags padrões

<!DOCTYPE html>

Esta tag diz ao navegador que este arquivo é html

<head>

Determina as informações gerais da página, como idioma, título, descrições do conteúdo, autor e etc...

<meta>

Define informações de metadados, ou seja, informações sobre a página mas que não estará na página, como o autor.
